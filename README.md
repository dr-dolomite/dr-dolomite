## 🚀 Featured Project — QuecManager

<p align="center"><i>Visualize, configure, and optimize a cellular modem from a clean web dashboard.</i></p>

**QuecManager** (successor to my earlier **SimpleAdmin**) is a full management suite for Quectel 4G/5G modems — a statically-exported **Next.js 16 / React 19 / TypeScript** frontend talking to a **POSIX-shell + CGI backend** running on OpenWRT.

- 📡 **Real-time monitoring** — live RSRP/RSRQ/SINR signal dashboard, latency & bandwidth charts over WebSocket, automatic network-event detection
- 🎛️ **Deep cellular control** — band / tower / frequency locking, APN & SIM profiles, cell scanner, IMEI tools, AT command terminal
- 🛡️ **Reliability built in** — a 4-tier connection watchdog (ifup → CFUN toggle → SIM failover → reboot), email downtime alerts, and in-app OTA updates
- 🏗️ **Scale** — ~185 React components, 38 custom hooks, 58 CGI endpoints, and 11 background daemons working together

It deploys through **[quectel-rgmii-toolkit](https://github.com/iamromulan/quectel-rgmii-toolkit)** — a collaborative project with [@iamromulan](https://github.com/iamromulan) for getting software onto the modem's Linux application processor.
